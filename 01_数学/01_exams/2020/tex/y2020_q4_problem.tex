\noindent\textbf{〔IV〕}
$\mathbb{R}^3$ における互いに独立な 3 つのベクトルを $\bm{a},\bm{b},\bm{c}$ とする.
$\bm{a}$ と $\bm{b}$ の外積を $\bm{a}\times\bm{b}$, $\bm{a}$ と $\bm{b}$ の内積を $(\bm{a},\bm{b})$ と
記号で書くとする. $(\bm{a}\times\bm{b},\bm{c})=(\bm{a},\bm{b}\times\bm{c})$ の関係が成り立つことを示せ.
また, $\bm{a}=\overrightarrow{OA},\ \bm{b}=\overrightarrow{OB},\ \bm{c}=\overrightarrow{OC}$ を
3 辺とする 4 面体の体積を, ベクトル $\bm{a},\bm{b},\bm{c}$ を用いて求めよ. ここで $O$ は原点とする.
