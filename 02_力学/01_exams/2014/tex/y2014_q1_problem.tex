\section*{第1問〔I〕}

質量 $m$ の質点が長さ $a$ の軽い糸につながれ，点 O を中心として反時計回りに
一定の角速度 $\omega$ で回転している。このとき，以下の問い (1)〜(4) に答えよ。
ただし，重力は無視する。

\begin{enumerate}[label=(\arabic*)]
  \item 質点の速度の大きさを求め，向きを図示せよ。
  \item 質点の加速度の大きさを求め，向きを図示せよ。
  \item 質点が 1 周する間に，糸の張力が質点にする仕事を求めよ。
  \item 質点の角運動量が保存されることを示せ。
\end{enumerate}
