\section*{第1問〔I〕}

$x$ 軸上を運動する質量 $m$ の質点に，復元力 $-m\omega^2 x$ と抵抗力 $-2m\gamma\dot{x}$ が
働く場合，(1)〜(4) に答えよ。ただし，$\dot{x}$ は速度を表し，$\omega$ と $\gamma$ は
正の定数とする。また，質点は時刻 $t=0$ のとき $x=L$ にあり $\dot{x}=0$ であった。

\begin{enumerate}[label=(\arabic*)]
  \item 質点に対する運動方程式の一般解を求めよ。
  \item $\omega>\gamma$ の場合，質点の運動の概形を描け。なお，横軸は時間，
        縦軸は質点の位置とする。
  \item $\omega<\gamma$ の場合，質点の運動の概形を描け（横軸は時間，縦軸は質点の位置）。
  \item $\omega>\gamma$ の場合，質点に復元力 $-m\omega^2 x$ と抵抗力 $-2m\gamma\dot{x}$ に
        加えて，外力 $mF_0\cos\omega_0 t$ が働くとき，時間が十分経過したときの質点の
        運動の様子を述べよ。ただし，$\omega_0$ と $F_0$ は正の定数である。
\end{enumerate}
