# 01_exams/ — 力学 過去問 転記の基準（引き継ぎ用）

AY2012 で施主承認を得た品質基準。**全年度この水準に揃える**こと。

## 成果物（問題のみ・解答は置かない）
- `01_exams/<年>/tex/y<年>_problems.tex` … 年度ラッパー（`\documentclass`＋`\input`）
- `01_exams/<年>/tex/y<年>_q1_problem.tex`, `..._q2_problem.tex`, …（大問ごと1ファイル）
- `01_exams/<年>/y<年>_problems.pdf` … コンパイル済みPDF（年度直下へ置く）

## 体裁・粒度（厳守）
1. **図は原図に忠実に全て再現する。** 矢印の向き・速度/角度ラベル・角度弧・
   開円(○)と塗円(●)の別・ばねコイル・座標軸と目盛まで写し取る。**図の欠落はNG。**
   作図は tikz（コイルは `decorations.pathmorphing` の `coil`）。
2. **大問が変わる箇所でページを切り替える。** ラッパーで大問の `\input` の間に
   `\clearpage` を入れ、各大問を独立ページから始める。
3. 問題文は原文を常体で正確に転記。判読困難な箇所は推測で埋めず `rem`（注意）環境で明記。
4. 小問は `enumerate[label=(\arabic*)]`。見出しは `\section*{第n問〔...〕}`。
5. preamble は `\usepackage{../../../00_common/preamble}`。コイル等で
   tikzlibrary が要るならラッパー側で `\usetikzlibrary{...}` を補う。

## 手順
1. `pdftoppm -png -r 200`（不鮮明なら `-r 300`）で原本を画像化し、全小問・全図を精読。
   要約を出して誤読を自己点検。
2. tex 作成 → 年度フォルダで `xelatex` を2回。**Overfull・警告ゼロ**まで直す
   （preamble 名の整合警告のみは無害で許容）。
3. 生成PDFを1ページずつ画像化し、原図と図・式・ページ送りを突き合わせて確認。
4. 中間物（`*.aux *.log`）を消し、PDF を年度直下へ移動。

## コミット
年度ごとに `exams: AY<年> 力学 問題転記`。担当外科目は読み書きしない。
