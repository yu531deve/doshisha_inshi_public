# CLAUDE.md — リポジトリ共通ガード

このリポジトリは **4科目（01_数学 / 02_力学 / 03_制御工学 / 04_材料力学）の独立した小プロジェクト**の集合。

## 絶対ルール
- 作業前に**担当科目を1つに固定**し、その `0X_<科目>/` に `cd` してから始める。
- **担当外の科目ディレクトリは読み書きしない**（混入事故の防止）。
- 各科目の詳細規約は、その科目フォルダ直下の `CLAUDE.md` に従う。
- 東大版（各科目 `参考/`）は**体裁・粒度の手本としてのみ**使い、分野分類・問題内容・命名・出所を持ち込まない。
- HELDOUT は無し（全年度が作成対象）。年度は AY（年度）表記で統一。

## 進め方
- まず1年度だけ仕上げて粒度を確定 →（手本として固定）→ 横展開。
- 1年度を仕上げてコミットした区切りで止める。コミット例: `solutions: AY2026 数学`。
